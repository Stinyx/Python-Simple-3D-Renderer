### Simple 3D renderer in python using the raylib library

- custom object loading using load_object(object_path) (object must be .obj file)
- object statistics with draw_object_statistics(object variable)
- move camera around with mouse
- player movement ***W***, ***A***, ***S***, ***D*** to move around the space and ***SPACE***, ***CTRL*** to go up and down

### How to run (VSCode):

- download the file and extract into a folder (or use git)
- open the folder in vscode
- make a venv in the terminal using: `python -m venv .venv`
- open the venv with: `.\.venv\Scripts\activate`
- download requirements: `pip install -r requirements.txt`
- run the main.py: `python main.py`
